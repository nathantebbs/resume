# Resume

This repository contains my resume(s) built with [Typst](https://typst.app/) using the [basic-resume](https://typst.app/universe/package/basic-resume) template.

## Building

To compile the resume to PDF:

```bash
typst compile resume.typ
```

To watch for changes and auto-compile:

```bash
typst watch resume.typ
```

## Structure

- `resume.typ` - Main resume using the basic-resume template
